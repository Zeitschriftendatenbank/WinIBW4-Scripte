Diese Seite bietet eine Übersicht der Scripte für die WinIBW4, die von der Zeitschriftendatenbank (ZDB) entwickelt und gepflegt werden.

Über das Projekt

Die WinIBW4-Scripte erleichtern die tägliche Arbeit in der WinIBW4-Oberfläche. Sie stehen zur freien Nutzung und Weiterentwicklung bereit und werden auf GitHub verwaltet.

Zur GitHub-Repository Issues & Anfragen

Mitmachen

Neue Scripte, Verbesserungen oder Fehlerkorrekturen können über GitHub als Pull Request eingereicht werden. Fragen und Anregungen sind im Issue-Tracker willkommen.
