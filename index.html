WinIBW4-Scripte

Übersicht der Scripte für die WinIBW4

Startseite | Über das Projekt
Alle Scripte für die WinIBW4 auf einen Blick. Klicken Sie auf „Download" um ein Script herunterzuladen, oder besuchen Sie das jeweilige GitHub-Repository für weitere Informationen.

Verfügbare Scripte

📦

WinIBW4-ZDB

JavaScript

ZDB-Scripte für die WinIBW4. Das zentrale Script-Paket der Zeitschriftendatenbank für die WinIBW4-Oberfläche.

⬇ Download GitHub Docs
📊

excelTool

JavaScript

WinIBW4 excelTool – ermöglicht den Export und die Verarbeitung von Daten aus der WinIBW4 in Excel-kompatible Formate.

⬇ Download GitHub
📥

csvImportTemplate

JavaScript

Template-Importtool für die WinIBW4 – erleichtert den strukturierten Import von CSV-Daten in die WinIBW4.

⬇ Download GitHub
🗂️

WinIBW-Set

JavaScript

Works perfectly on WinIBW-Sets – Werkzeuge zur komfortablen Arbeit mit Treffermengen (Sets) in der WinIBW.

⬇ Download GitHub
🪵

WinIBW-LOGGER

JavaScript

Logger für die WinIBW – protokolliert Aktionen und Ereignisse in der WinIBW zur Fehleranalyse und Nachvollziehbarkeit.

⬇ Download GitHub
🔔

WinIBW-notify

JavaScript

Notify instead of messageBox – ersetzt störende modale Dialogboxen durch moderne, nicht-blockierende Benachrichtigungen.

⬇ Download GitHub
🧪

WinIBW-testrunner

JavaScript

Test-Framework für WinIBW4-Scripte – ermöglicht das automatisierte Testen von Scripten direkt in der WinIBW4-Umgebung.

⬇ Download GitHub
🔧

winibw-misc

JavaScript Intern

Frequently used functions for the WinIBW – eine Sammlung häufig genutzter Hilfsfunktionen als gemeinsame Basis für andere Scripte.

⬇ Download GitHub
📡

WinIBW-CSV-API

JavaScript

CSV-API script for WinIBW3 – stellt eine API bereit, die Daten aus der WinIBW als CSV exportiert und weiterverarbeitet.

⬇ Download GitHub
